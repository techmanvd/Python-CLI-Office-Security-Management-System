# Office Security System (CLI)

A lightweight, command-line interface (CLI) tool built in Python to manage office visitor logs and security alerts. This project simulates a front-desk security system using Object-Oriented Programming (OOP).

    * Review a history of issued alerts.
* **Interactive Menu:** A continuous loop menu that allows multiple operations without restarting the program.
